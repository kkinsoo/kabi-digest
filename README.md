## 🗂 About kabi-digest

**kabi-digest** gathers the top discussions and news from two popular platforms: V2EX and Hacker News. It creates a simple, easy-to-read daily report for users who want to catch up on tech and community highlights without browsing multiple sites.  

This application runs on Windows computers and outputs a digest you can read on any device.

---

## ⚙️ What it Does

- Collects trending posts from V2EX and Hacker News every day.
- Summarizes the main points and lists links for deeper reading.
- Organizes content into clear sections for easy navigation.
- Saves the daily digest as a text or HTML file you can open anytime.

---

## 💻 System Requirements

To run kabi-digest, your Windows computer needs:

- Windows 10 or newer.
- At least 4 GB of RAM.
- 100 MB of free disk space.
- Internet connection to fetch the latest posts.

No special hardware or software installations are needed.

## 🔧 Basic Troubleshooting

If the app does not run:

- Make sure you downloaded the right file for Windows (`.exe` or `.zip`).
- Confirm your computer meets the system requirements.
- Check your internet connection.
- If the program shows an error, try re-downloading the app.
- Run the app as an administrator by right-clicking the file and choosing "Run as administrator."

If you still have issues, try restarting your computer and running the app again.

## 📅 Scheduling Daily Use (Optional)

You can set your computer to run kabi-digest automatically every day.

1. Use Windows Task Scheduler.
2. Create a new task.
3. Set it to run the `kabi-digest.exe` at a fixed time daily.
4. Ensure your computer is on and connected to the internet at that time.

This step is optional but useful if you want new digests without manual work.
